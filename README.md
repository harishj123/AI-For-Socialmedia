# 🚨 SilentShield

## AI-Based Early Emotional Distress & Risk Detection System

---

## 📌 Overview

SilentShield is an AI-powered social impact solution that detects early signs of emotional distress and potential domestic abuse through text-based sentiment and risk classification.

The system analyzes user input using Machine Learning and classifies emotional state into:

* 🟢 Low Risk
* 🟡 Medium Risk
* 🔴 High Risk

The goal is to provide early awareness and proactive intervention support before situations escalate.

---

## 🎯 Problem Statement

Victims of emotional distress and domestic abuse often cannot openly seek help.

Traditional emergency systems work only after explicit reporting. There is no proactive AI-based mechanism that identifies distress signals early and suggests preventive action.

This leads to:

* Delayed intervention
* Escalation of abuse
* Mental health deterioration
* Preventable harm

SilentShield addresses this gap through AI-driven early detection.

---

## 💡 Our Solution

SilentShield uses:

✔ Natural Language Processing (NLP)
✔ TF-IDF Text Vectorization
✔ Random Forest Classification
✔ Real-time Risk Scoring

The application analyzes user messages and predicts emotional risk levels along with probability scores.

---

## 🧠 How It Works

1. User enters a message in the interface
2. Text is transformed using TF-IDF vectorization
3. A trained ML classifier predicts distress level
4. Risk probability scores are displayed
5. System suggests appropriate safety action

## 🚀 How to Run the Project

Clone the repository:

```bash
git clone https://github.com/yourusername/SilentShield.git
cd SilentShield
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

The application will open in your browser automatically.
